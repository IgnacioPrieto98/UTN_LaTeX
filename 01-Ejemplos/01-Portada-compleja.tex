% Anteriormente, en el ejemplo 01-Portada simple se realizó una
% portada la cual presenta características primitivas de un 
% documento. Aun así, puede resultar necesario presentar
% más información en la portada.
% 
% Por este motivo se recurre a un entorno denominado pagina de 
% título (titlepage) en el cual se define una hoja que posee
% todo el contenido deseado.
% 
 
% Para poder crear dicho entorno se abre un entorno con el comando
% \begin{titlepage} y se lo cierra con el comando \end{titlepage}.
% Dentro de este se especifica el contenido que se quiere presentar.
%
