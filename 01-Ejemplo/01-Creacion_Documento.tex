% Para empezar un documento se debe definir una estructura sobre la 
% cual se va a trabajar. Estas estructuras se denominan clases.
% Las clases más usuales son:
%	-book
%	-article
%	-letter
%	-slides
%	-report

% El siguiente comando permite definir que tipo de clase posee el doc.
\documentclass{article}

% Luego de definir la clase del documento se debe definir un entorno en
% el cual se desarrolla todo lo qeu va dentro del documento. Este entorno
% se define mediante un comienzo "\begin{document}" y un final "\end{document}".
% Entender el concepto de entorno resulta importante debido a que LaTeX es un 
% lenguaje de de marcado que trabaja mucho con ellos. 

% Creación de entorno de documento.
\begin{document}

% Dentro del entorno de documento se presenta el texto que se quiere presentar.

Mi primer texto en LaTeX.


\end{document}
